How this works

This page fetches .agents/TASKS.json and the routine manifests directly. It must be served over HTTP (most browsers block file:// XHR). Quick options:

  • From repo root: npx serve . or python -m http.server 8000
  • Then visit http://localhost:8000/factory-status.html
  • Or run the CLI fallback: node factory-status.mjs

For a frozen snapshot, run node factory-status.mjs --json and pipe the output into your monitoring tool of choice.
